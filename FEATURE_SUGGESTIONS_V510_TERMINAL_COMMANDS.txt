SUSTAINABLE CATALYST FEATURE SUGGESTIONS v5.1.0
Integrated Knowledge Base and Documentation Library

WORDPRESS UPGRADE
-----------------
1. Back up WordPress.
2. Open Plugins → Add New → Upload Plugin.
3. Upload sustainable-catalyst-feature-suggestions-v5.1.0-wordpress.zip.
4. Choose Replace current with uploaded.
5. Open Feature Suggestions → Knowledge Base Library.
6. Confirm 96 articles, 16 products, and 32 samples.
7. Select Import or Repair Documentation if automatic migration has not completed.
8. Confirm Support → Knowledge Base shows the expandable product directory.
9. Deactivate the older standalone Sustainable Catalyst Support Knowledge Base content-pack plugin after validation.
10. Clear caches. Save Settings → Permalinks once only if an article URL returns 404.

GITHUB INSTALL AND PUSH
-----------------------
cd ~/Downloads
chmod +x install_and_push_sustainable_catalyst_feature_suggestions_v5_1_0.sh
./install_and_push_sustainable_catalyst_feature_suggestions_v5_1_0.sh

LOCAL VALIDATION
----------------
cd /path/to/sustainable-catalyst-feature-suggestions
find wordpress tests -type f -name '*.php' -print0 | xargs -0 -n1 php -l
for f in tests/test-*.php; do php "$f" || exit 1; done
find wordpress/sustainable-catalyst-feature-suggestions/assets -type f -name '*.js' -print0 | xargs -0 -n1 node --check
PYTHONPATH=backend python3 -m pytest backend/tests -q
