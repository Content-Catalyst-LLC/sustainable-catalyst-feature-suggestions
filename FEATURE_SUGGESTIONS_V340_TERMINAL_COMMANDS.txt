FEATURE SUGGESTIONS v3.4.0 — INSTALL, VALIDATE, COMMIT, AND PUSH

1. Place these files in ~/Downloads:
   - sustainable-catalyst-feature-suggestions-v3.4.0-repo.zip
   - install_and_push_sustainable_catalyst_feature_suggestions_v3_4_0.sh

2. Run:

cd ~/Downloads

SCRIPT="$(ls -t install_and_push_sustainable_catalyst_feature_suggestions_v3_4_0*.sh | head -1)"

chmod +x "$SCRIPT"
bash "$SCRIPT"

The installer uses Python 3.12 or 3.13, installs the pinned FastAPI dependencies including httpx, starts from a fresh GitHub clone, runs all PHP and Python validation, commits the release, rebases on any newer remote changes, and pushes main.

If Python 3.12 or 3.13 is missing:

brew install python@3.13

Then rerun the block above.
