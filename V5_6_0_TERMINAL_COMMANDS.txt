Sustainable Catalyst Product Support and Feedback Platform v5.6.0
Feedback Intelligence and Product Signals

1. Place these files in ~/Downloads:
   - sustainable-catalyst-product-support-feedback-platform-v5.6.0-release-bundle.zip
   - install_and_push_v5_6_0_macos.sh

2. Run:

cd ~/Downloads
chmod +x install_and_push_v5_6_0_macos.sh
./install_and_push_v5_6_0_macos.sh

The installer will:
- choose Python 3.13 or 3.12 and reject Python 3.14
- locate the newest v5.6.0 release bundle or repository ZIP
- clone the current GitHub main branch
- create a safety backup in ~/Downloads
- install the complete v5.6.0 repository
- create an isolated Python validation environment
- enforce a binary pydantic-core wheel
- run the complete v5.6.0 validation suite
- run Git whitespace validation
- commit v5.6.0
- create the v5.6.0 tag
- push main and the tag

When no compatible Python is installed:

brew install python@3.13

Then rerun the installer.
