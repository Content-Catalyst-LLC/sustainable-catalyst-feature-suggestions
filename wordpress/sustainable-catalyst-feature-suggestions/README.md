# Sustainable Catalyst Product Support and Feedback Platform

## Current release: v6.2.0

Version 6.2.0 adds the private Help Desk Case Foundation while preserving the public Support Center, publication library, Known Issues, release intelligence, feedback signals, analytics, public integrations, and cross-product handoffs.

Primary public shortcode: `[scfs_product_support_center]`

Connected platform shortcode: `[scfs_connected_product_support_platform]`

Dedicated Knowledge Base shortcode: `[scfs_support_knowledge_base]`

Unified search shortcode: `[scfs_unified_support_search]`

Issue and release intelligence shortcode: `[scfs_issue_release_intelligence]`

Cross-product support graph shortcode: `[scfs_cross_product_support_graph]`

Product support embed shortcode: `[scfs_support_embed product="decision-studio"]`

The connected platform is managed in **Support & Feedback → Connected Platform**.

The private Help Desk is managed in **Support & Feedback → Help Desk Cases**. There is no public case shortcode or public case API.

The plugin directory, text domain, PHP class, `scfs_*` identifiers, post types, REST namespace, article URLs, options, and existing records remain backward compatible.
