# Sustainable Catalyst Feature Suggestions

A lightweight WordPress plugin for collecting structured platform feature suggestions.

## Shortcode

```text
[sustainable_catalyst_feature_suggestions]
```

## What it does

- Renders a structured feature suggestion form
- Saves submissions as private WordPress records
- Sends the site admin an email notification
- Adds a Feature Suggestions admin menu
- Provides CSV export from the WordPress admin
- Includes a non-confidential submission notice
- Uses nonce verification, sanitization, and a honeypot field

## Suggested page

Use the shortcode on:

```text
/platform/feature-suggestions/
```

## Submission categories

- New platform module
- Online demo improvement
- Research Librarian feature
- GitHub repository improvement
- Data or export feature
- Article map or library suggestion
- Accessibility or usability issue
- Bug report
- Documentation improvement
- Consulting or workflow idea
- Other

## Boundary note

The form is for non-confidential suggestions only. Visitors should not submit confidential, proprietary, sensitive personal, medical, legal, financial, or regulated information.
