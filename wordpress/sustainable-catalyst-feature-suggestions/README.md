Version 6.0.0 establishes the Connected Product Support and Feedback Platform across the Support Center, publication library, Known Issues, release intelligence, feedback signals, analytics, public integrations, and cross-product handoffs.

Primary public shortcode: `[scfs_product_support_center]`

Connected platform shortcode: `[scfs_connected_product_support_platform]`

Cross-product support graph shortcode: `[scfs_cross_product_support_graph]`

The plugin directory, text domain, PHP class, `scfs_*` identifiers, post types, REST namespace, article URLs, options, and existing records remain backward compatible.
