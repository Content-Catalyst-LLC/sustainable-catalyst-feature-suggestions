# Sustainable Catalyst Product Support and Feedback Platform

## Current release: v6.3.0

Version 6.3.0 adds the secure Customer Support Portal and participant-visible conversations while preserving the public Support Center, publication library, Known Issues, release intelligence, private case foundation, Agent Workspace, feedback signals, analytics, and integrations.

Primary public shortcode: `[scfs_product_support_center]`

Connected platform shortcode: `[scfs_connected_product_support_platform]`

Product support embed shortcode: `[scfs_support_embed product="decision-studio"]`

Customer portal shortcode: `[scfs_help_desk_customer_portal]`

The connected platform is managed in **Support & Feedback → Connected Platform**.

The private Help Desk is managed in **Support & Feedback → Help Desk Cases** and **Agent Workspace**. The secure customer portal is rendered only for a valid token-exchanged session and does not expose a public case list API.

The plugin directory, text domain, PHP class, `scfs_*` identifiers, post types, REST namespace, article URLs, options, and existing records remain backward compatible.


## v6.3.0
