Product Support and Feedback Platform v6.1.0 — Terminal Commands

1. Place these files in ~/Downloads:
   - sustainable-catalyst-product-support-feedback-platform-v6.1.0-release-bundle.zip
   - install_and_push_v6_1_0_macos.sh

2. Run:

cd ~/Downloads
chmod +x install_and_push_v6_1_0_macos.sh
./install_and_push_v6_1_0_macos.sh

Optional archive-selection preflight:

cd ~/Downloads
SCFS_PREFLIGHT_ONLY=1 ./install_and_push_v6_1_0_macos.sh

Manual validation after extraction:

cd ~/Downloads/sustainable-catalyst-feature-suggestions
PYTHON_BIN=/opt/homebrew/bin/python3.13 bash ./validate_v6_1_0.sh

Help Desk schema checks after WordPress activation:

wp scfs help-desk schema
wp scfs help-desk integrity

Synthetic case creation example:

wp scfs help-desk create \
  --subject="Decision brief export is unavailable" \
  --description="The supported PDF export workflow returns an error." \
  --requester-ref="contact-engagement:inquiry-184" \
  --product=decision-studio \
  --version=2.0.1 \
  --component=briefing \
  --type=unexpected_behavior \
  --priority=p2_high \
  --severity=major \
  --consent=recorded

Review a case:

wp scfs help-desk case SC-2026-000184

Transition a case:

wp scfs help-desk transition SC-2026-000184 resolved \
  --reason="Verified recovery steps completed."
