Sustainable Catalyst Product Support and Feedback Platform v5.2.7
Support Center Production Integration and Interface Hardening

Place these files in ~/Downloads:
- sustainable-catalyst-product-support-feedback-platform-v5.2.7-release-bundle.zip
- install_and_push_v5_2_7_macos.sh

Run:

cd ~/Downloads
chmod +x install_and_push_v5_2_7_macos.sh
./install_and_push_v5_2_7_macos.sh

The installer will:
- select Python 3.13 or 3.12 and reject Python 3.14;
- locate the v5.2.7 release bundle or repository ZIP;
- clone the current GitHub main branch;
- create a timestamped safety backup in ~/Downloads;
- install the complete v5.2.7 source tree;
- build an isolated validation environment;
- require a binary pydantic-core wheel;
- run the WordPress, JavaScript, CSS, JSON, Python, FastAPI, manifest, and whitespace suites;
- commit and push the validated release.
