Sustainable Catalyst Product Support and Feedback Platform v5.4.0
Known Issues and Release Intelligence Integration

INSTALL AND PUSH

1. Place these files in ~/Downloads:
   sustainable-catalyst-product-support-feedback-platform-v5.4.0-release-bundle.zip
   install_and_push_v5_4_0_macos.sh

2. Run:

   cd ~/Downloads
   chmod +x install_and_push_v5_4_0_macos.sh
   ./install_and_push_v5_4_0_macos.sh

The installer will:
- select Python 3.13 or 3.12 and reject Python 3.14;
- locate the newest v5.4.0 release bundle or repository ZIP;
- validate the release manifest and governance boundaries;
- clone the current GitHub main branch;
- create a safety backup in ~/Downloads;
- install the complete v5.4.0 source tree;
- create an isolated Python validation environment;
- run the PHP, JavaScript, JSON, Python, CSS, FastAPI, and compatibility suites;
- run git diff --check;
- commit, tag, and push the validated release.

DIRECT WORDPRESS INSTALLATION

Upload this file through WordPress Plugins > Add New > Upload Plugin:

   sustainable-catalyst-feature-suggestions-v5.4.0-wordpress.zip

The repository installer is preferred because it validates and preserves the complete source release.
