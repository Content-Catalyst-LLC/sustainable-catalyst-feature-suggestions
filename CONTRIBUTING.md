# Contributing

This repository supports the Sustainable Catalyst feature suggestion workflow.

Useful contributions include:

- Improving the WordPress plugin form UX.
- Improving validation, spam protection, or export quality.
- Improving accessibility and keyboard navigation.
- Improving admin review workflows.
- Improving documentation and examples.
- Adding tests or additional linting.

Do not commit production exports, user-submitted private records, confidential information, credentials, API keys, analytics secrets, or sensitive personal information.
