# Feedback Intelligence Dashboard

Version 2.3.0 adds an administrator-only dashboard under **Feature Suggestions → Intelligence**. It aggregates WordPress workflow data and stored AI triage results without sending additional data to an AI provider.

## Views

- submission and analysis coverage
- workflow status and category distributions
- platform-area and feature-type routing
- top topics, sentiment, and suggested actions
- filtered advisory opportunity ranking
- CSV export and protected REST output

## Governance

AI fields are advisory. The dashboard never automatically approves, rejects, publishes, scores, or schedules a suggestion. Safety flags are counts only; reviewers must inspect the original record.
