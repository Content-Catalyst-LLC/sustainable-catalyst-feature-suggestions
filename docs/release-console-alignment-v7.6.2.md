# Release Console Alignment v7.6.2

v7.6.2 preserves the shared responsive product grid and adds operational cache and footer verification.

- Product, Version, State, and Source remain on one shared responsive grid.
- Intelligence badges remain beneath product names.
- Repository and Support links remain in the fixed footer only.
- The repository destination resolves from the explicit footer setting or the canonical Product Support and Feedback GitHub repository.
- The Support destination resolves from its saved footer setting.
- Footer and registry changes invalidate the Release Console cache epoch.
- Legacy terminal, blackboard, compact, and directory layouts remain supported.
