# GitHub Connection v7.6.0

Open **Support & Feedback → GitHub Connection**.

From one screen you can save the encrypted GitHub token, test mapped repositories, synchronize one or all repositories, see automatic-sync health, and edit the Release Console repository and support links.

Published GitHub Releases are preferred. When no release is published, the console can use the newest semantic version tag. Public repositories can be tested without a token; private repositories require the saved token or a server-level override.
