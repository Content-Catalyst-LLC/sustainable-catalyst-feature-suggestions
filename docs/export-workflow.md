# Export Workflow

The plugin includes an admin CSV export screen under:

```text
Feature Suggestions → Export CSV
```

The v2 export includes:

- ID
- Date
- Post status
- Title
- Category
- Priority
- Review status
- Impact score
- Effort score
- Roadmap area
- GitHub issue URL
- Problem
- Suggested feature
- Success criteria
- Beneficiaries
- Implementation notes
- Relevant URL
- Name
- Email
- Follow-up permission
- Consent status
- Referrer
- IP hash
- Admin notes

The export is intended for private review, backlog planning, and GitHub issue creation. Do not commit real exports to this repository.

The `exports/` directory includes only a `.gitkeep` placeholder so the repository documents the export workflow without storing visitor data.
