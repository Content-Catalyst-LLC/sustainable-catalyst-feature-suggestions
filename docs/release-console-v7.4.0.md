# Release Console v7.4.0

`[sc_release_board]` continues to render the compact rotating Release Console introduced in v7.3.2 and hardened in v7.3.3.

v7.4.0 changes the data authority rather than the visual contract. Each product is placed using its governed `console_screen` field. Retired and superseded records are hidden by default, while all product names remain non-navigating console labels. Release and Support remain the only footer links.
