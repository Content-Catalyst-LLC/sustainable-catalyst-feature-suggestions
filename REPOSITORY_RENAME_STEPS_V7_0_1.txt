REPOSITORY RENAME STEPS — v7.0.1

1. Confirm Product Support and Feedback Platform v7.0.0 is committed and tagged.
2. In GitHub, open the repository Settings page.
3. Rename the repository from:
   sustainable-catalyst-feature-suggestions
   to:
   sustainable-catalyst-product-support-feedback
4. Do not create another repository using the old name.
5. Download the v7.0.1 release bundle to ~/Downloads.
6. Run:

   cd ~/Downloads
   chmod +x install_and_push_sustainable_catalyst_product_support_feedback_v7_0_1_macos.sh
   ./install_and_push_sustainable_catalyst_product_support_feedback_v7_0_1_macos.sh

The installer will update the Git remote and safely rename an existing local repository folder.
It will not rename the WordPress plugin folder.
