# Sustainable Catalyst Product Support and Feedback Platform v7.3.0

## Release Blackboard Shortcode

v7.3.0 introduces the public `[sc_release_board]` shortcode backed by the canonical product registry.

### Included

- Automatic installed WordPress plugin versions and manual product versions in one governed board
- Foundation, research and intelligence, data and analysis, creation and systems, and commercial groups
- Blackboard, compact, and directory layouts
- Product, group, status, visibility, and display-limit controls
- Manual Catalyst Intelligence display without exposing its private repository
- Semantic list markup, visible status text, responsive layouts, and reduced-motion support
- Cache invalidation after product-registry saves and plugin-discovery completion
- Cache-safe stylesheet versioning
- FastAPI release-board projection validation

### Compatibility

No database migration is required. The WordPress plugin directory, text domain, REST namespace, post types, options, and existing shortcodes remain unchanged.
