Feature Suggestions v4.4.0 — Install, validate, commit, and push

1. Place these files in ~/Downloads:
   - sustainable-catalyst-feature-suggestions-v4.4.0-repo.zip
     or sustainable-catalyst-feature-suggestions-v4.4.0-release-bundle.zip
   - install_and_push_sustainable_catalyst_feature_suggestions_v4_4_0.sh

2. Run:

cd ~/Downloads
brew install python@3.13
hash -r
SCRIPT="$(ls -t install_and_push_sustainable_catalyst_feature_suggestions_v4_4_0*.sh | head -1)"
chmod +x "$SCRIPT"
bash "$SCRIPT"

3. For direct WordPress installation, upload:

sustainable-catalyst-feature-suggestions-v4.4.0-wordpress.zip

4. After activation, open:

Feature Suggestions → Product Reliability
