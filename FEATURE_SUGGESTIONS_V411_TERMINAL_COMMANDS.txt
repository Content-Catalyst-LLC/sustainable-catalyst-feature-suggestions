Feature Suggestions v4.1.1 — install, validate, commit, and push

1. Put these files in ~/Downloads:
   - sustainable-catalyst-feature-suggestions-v4.1.1-repo.zip
     OR sustainable-catalyst-feature-suggestions-v4.1.1-release-bundle.zip
   - install_and_push_sustainable_catalyst_feature_suggestions_v4_1_1.sh

2. Run:

cd ~/Downloads

brew install python@3.13
hash -r

SCRIPT="$(ls -t install_and_push_sustainable_catalyst_feature_suggestions_v4_1_1*.sh | head -1)"
chmod +x "$SCRIPT"
bash "$SCRIPT"

For direct WordPress installation, upload:

sustainable-catalyst-feature-suggestions-v4.1.1-wordpress.zip

After activation, open:

Feature Suggestions → Content Operations

Recent successful imports can be rolled back from the Recent imports and recovery table. Rollback moves imported records to WordPress Trash; it does not permanently delete them.
