# Product Support and Feedback Platform v6.0.0

## Connected Product Support and Feedback Platform

v6.0.0 completes the transition from a feature-suggestion plugin into a connected support, documentation, operational-intelligence, feedback, analytics, and integration platform.

### Added

- Five-layer connected platform contract.
- Connected product support dossiers.
- Platform health, readiness, and integrity summaries.
- Connected support journey planning.
- Public connected-platform shortcode and compatibility alias.
- Connected Platform administration screen.
- WordPress REST and FastAPI parity.
- Daily connected-platform snapshots and JSON export.
- WP-CLI status, product, journey, and refresh commands.
- Responsive and print-safe public interface.
- Versioned JSON schema and example records.

### Governance

Specialist modules remain the source of truth. v6.0.0 does not automatically publish content, resolve Known Issues, change release or roadmap state, redirect users, or create private support cases.

### Compatibility

The plugin slug, `scfs_*` identifiers, CPTs, taxonomies, shortcodes, REST namespace, database records, settings, Support Article URLs, `/support/` route, and legacy Knowledge Base redirects remain compatible. No database migration is required.

### Distribution repair R2

The macOS validation path no longer uses the Bash 4-only `mapfile` command. CSS discovery now runs inside the Python structural-validation block, preserving full validation compatibility with the default macOS Bash 3.2 shell. Product runtime behavior is unchanged.
