SUSTAINABLE CATALYST FEATURE SUGGESTIONS v5.1.0
Integrated Knowledge Base and Documentation Library

BEFORE UPGRADING
----------------
1. Back up WordPress and the current Feature Suggestions repository.
2. Keep the existing Feature Suggestions plugin active during replacement.
3. The older standalone Sustainable Catalyst Support Knowledge Base content-pack plugin can remain active during the upgrade, but deactivate it after the integrated Knowledge Base is confirmed.

WORDPRESS INSTALLATION
----------------------
1. Open WordPress Admin → Plugins → Add New → Upload Plugin.
2. Upload sustainable-catalyst-feature-suggestions-v5.1.0-wordpress.zip.
3. Choose Replace current with uploaded.
4. Activate the plugin if WordPress does not keep it active.
5. Open Feature Suggestions → Knowledge Base Library.
6. Confirm the validation summary shows:
   - 96 bundled articles
   - 16 products
   - 32 sample files
7. The upgrade normally imports the corpus automatically. If published records remain below 96, select Import or Repair 96 Guides.
8. Open the public Support page and select Knowledge Base.
9. Expand Browse the Knowledge Base and confirm product and section folders appear.
10. Open several guides and verify breadcrumbs, sample downloads, related guides, previous/next navigation, printing, and the usefulness rating form.
11. Deactivate the older standalone Sustainable Catalyst Support Knowledge Base content-pack plugin.
12. Clear WordPress, host, and Cloudflare caches.
13. Save Settings → Permalinks once only if a Support Article URL returns 404.

EXPECTED MIGRATION BEHAVIOR
---------------------------
- Missing integrated guides are created and published.
- Existing generated guides are updated.
- Manually edited article bodies are preserved.
- Product, version, component, issue type, article type, and collection assignments are repaired.
- The migration can be run repeatedly without creating duplicates.
- CSV and JSON samples are delivered directly from the plugin and are not uploaded to the Media Library.
- Knowledge Base navigation remains visible even if the article count is temporarily zero.

USEFULNESS RATINGS
------------------
Visitors can rate each article Yes or Not yet. They may optionally select a reason and leave a brief non-sensitive improvement note. Existing privacy-minimized duplicate protection and documentation-intelligence reporting remain active.

GITHUB INSTALL AND PUSH
-----------------------
Place the repository ZIP or release bundle and the installer in ~/Downloads, then run:

cd ~/Downloads
chmod +x install_and_push_sustainable_catalyst_feature_suggestions_v5_1_0.sh
./install_and_push_sustainable_catalyst_feature_suggestions_v5_1_0.sh

The script validates PHP, WordPress contracts, JavaScript, JSON, the Knowledge Base corpus, Python/FastAPI tests, archive structure, and secrets before committing and pushing main.
