# Product Support and Feedback Platform v6.7.0

## Workflow Automation and Operational Rules

v6.7.0 adds a governed workflow layer to the private help desk. Event-driven rules evaluate case context and produce auditable action plans. Low-risk reminders and internal review tasks may be scheduled automatically; assignments, status or priority changes, customer communication, closure, resolution, and external webhooks remain approval-controlled.

### Included

- Seven additive private workflow tables
- Case-event trigger boundary
- Ordered rules and bounded conditions
- Risk-classified action plans
- Approval queue and action integrity hashes
- Response templates with variable allowlists
- Customer-safe draft governance
- Agent macros with per-action approvals
- Reminder and follow-up scheduling
- Agent Workspace workflow panel
- Private REST and WP-CLI operations
- FastAPI planning and validation parity

No existing public or private record migration is required.
