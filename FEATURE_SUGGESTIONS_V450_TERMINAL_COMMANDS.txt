Feature Suggestions v4.5.0 — Install, validate, commit, and push

Place these files in ~/Downloads:
- sustainable-catalyst-feature-suggestions-v4.5.0-repo.zip
  or sustainable-catalyst-feature-suggestions-v4.5.0-release-bundle.zip
- install_and_push_sustainable_catalyst_feature_suggestions_v4_5_0.sh

Run:

cd ~/Downloads
brew install python@3.13
hash -r
SCRIPT="$(ls -t install_and_push_sustainable_catalyst_feature_suggestions_v4_5_0*.sh | head -1)"
chmod +x "$SCRIPT"
bash "$SCRIPT"

For direct WordPress installation, upload:
sustainable-catalyst-feature-suggestions-v4.5.0-wordpress.zip

After activation, open:
Feature Suggestions → Cross-Product Support
