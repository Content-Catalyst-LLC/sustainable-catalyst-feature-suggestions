# Sustainable Catalyst Feature Suggestions v5.2.1

## Support Page Library Interface and Automatic Knowledge Base Rendering

This release transforms the public Support area into a library-style documentation experience while preserving the publication-quality article reader.

### Public documentation library

- Category-first landing view modeled on the Knowledge Library browsing pattern.
- Expandable documentation collections generated from published Support Articles.
- Product-first alternate view with complete generated documentation sets.
- Article rows include product, summary, verified version, reading time, and last-updated metadata.
- Search and product filtering remain first-class entry points.

### Expansion behavior

- Categories, products, and sections are fully expanded on first render.
- Expand all and Collapse all controls remain available.
- The selected category/product view is remembered for the browser session.
- Print output expands all documentation collections.

### Compatibility

- WordPress and Knowledge Base modules advance to v5.2.1.
- The unchanged FastAPI service remains compatible at v5.1.0.
- Existing Support Articles, taxonomies, URLs, imports, sample files, article styling, feedback, and integrations are preserved.
