# Product Support and Feedback Platform v7.7.1

## Plugin Discovery Intelligence

- Classifies all standard, site-active, network-active, inactive, must-use, and drop-in plugins.
- Ranks canonical product suggestions and exposes the matching signals and confidence.
- Adds bulk high-confidence mapping and bulk ignore workflows with safe skip behavior.
- Detects duplicate mappings, plugin-header identity conflicts, and repository mismatches.
- Compares installed plugin versions with governed GitHub release or semantic-tag versions.
- Adds searchable and filterable complete plugin inventory administration.
- Preserves manual confirmation, aliases, active-plugin connections, legacy shortcodes, accessibility, and reduced-motion behavior.
