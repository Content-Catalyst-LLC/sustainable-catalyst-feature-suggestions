# Sustainable Catalyst Feature Suggestions v5.1.1

## Knowledge Base Full-Width Reading Experience

This maintenance release makes the integrated Knowledge Base feel like a dedicated documentation system rather than a blog post layout.

### Changes

- Knowledge Base article and archive pages now request Astra's `no-sidebar` layout.
- The Publications navigation sidebar is removed from Knowledge Base views.
- The primary content column occupies the full page width.
- Article shells expand to 1320px while paragraph text remains constrained for readability.
- Tables, related guides, navigation cards, and documentation directories can use the wider canvas.
- Mobile layouts retain comfortable padding and responsive headings.

### Compatibility

The change is scoped to `sc_support_article` single pages and the Support Knowledge Base archive. Publications and sidebars elsewhere on the site are unchanged.
