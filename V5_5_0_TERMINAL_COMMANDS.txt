cd /path/to/sustainable-catalyst-feature-suggestions

WORDPRESS ADMIN

Open:
Support & Feedback → Content Operations

WP-CLI

wp scfs content-governance scan
wp scfs content-governance queue --state=overdue
wp scfs content-governance queue --priority=high
wp scfs content-governance verify 812 --note="Verified against the current supported release."

REST API

GET  /wp-json/scfs/v1/content-governance/schema
GET  /wp-json/scfs/v1/content-governance/queue
GET  /wp-json/scfs/v1/content-governance/record/812
POST /wp-json/scfs/v1/content-governance/scan
POST /wp-json/scfs/v1/content-governance/bulk
POST /wp-json/scfs/v1/content-governance/verify/812

The queue, record, scan, bulk, and verification routes require administrative capabilities.
