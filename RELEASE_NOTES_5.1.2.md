# Sustainable Catalyst Feature Suggestions v5.1.2

## Knowledge Base Publication-Parity Article Experience

- Reuses the visual language of Sustainable Catalyst publication articles for Knowledge Base guides.
- Replaces the theme-generated article heading with a first-party editorial masthead.
- Preserves the full-page, no-sidebar Knowledge Base layout.
- Adds publication-style typography, cream reference panels, maroon rules, editorial tables, code blocks, related guides, and article sequence navigation.
- Keeps support-specific product, version, audience, sample, verification, and feedback information.
- Expands the Knowledge Base directory, every product folder, and every documentation section by default.
- Retains visitor-controlled Expand all and Collapse all actions.
- Improves mobile and print presentation.
