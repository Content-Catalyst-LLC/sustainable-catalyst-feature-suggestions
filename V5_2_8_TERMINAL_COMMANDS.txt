Sustainable Catalyst Product Support and Feedback Platform v5.2.8
Support Article Content Integrity and Publication Validation

1. Download these two files into ~/Downloads:

   sustainable-catalyst-product-support-feedback-platform-v5.2.8-release-bundle.zip
   install_and_push_v5_2_8_macos.sh

2. Run:

   cd ~/Downloads
   chmod +x install_and_push_v5_2_8_macos.sh
   ./install_and_push_v5_2_8_macos.sh

The installer:

- selects Python 3.13 or 3.12 and rejects Python 3.14;
- creates a safety backup of the current GitHub repository;
- installs the complete v5.2.8 source;
- creates an isolated validation environment;
- runs PHP, WordPress contract, JavaScript, JSON, Python, CSS, FastAPI, manifest, and whitespace validation;
- commits the validated release;
- pushes the commit to GitHub main.

If no compatible Python is found:

   brew install python@3.13

Then rerun the installer.
