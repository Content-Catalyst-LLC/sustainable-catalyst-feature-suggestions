SUSTAINABLE CATALYST FEATURE SUGGESTIONS v3.1.0
================================================

WORDPRESS UPGRADE
-----------------
1. In WordPress Admin, open Plugins → Add New → Upload Plugin.
2. Upload:
   sustainable-catalyst-feature-suggestions-v3.1.0-wordpress.zip
3. Choose Replace current with uploaded if WordPress asks.
4. Activate the plugin if needed.
5. Open Feature Suggestions → Products & Integration.
6. Confirm the taxonomy migration status and coverage.

OPTIONAL WP-CLI MIGRATION
-------------------------
cd /path/to/wordpress
wp scfs migrate-product-taxonomies --batch=200

LOCAL VALIDATION
----------------
cd /path/to/sustainable-catalyst-feature-suggestions
find wordpress/sustainable-catalyst-feature-suggestions -name '*.php' -print0 | xargs -0 -n1 php -l
php tests/test-v310-structure.php
php tests/test-v310-bootstrap.php
PYTHONPATH=backend python3 -m pytest backend/tests -q

GITHUB UPDATE WITH THE INCLUDED SCRIPT
--------------------------------------
cd ~/Downloads
chmod +x install_and_push_sustainable_catalyst_feature_suggestions_v3_1_0.sh
bash ./install_and_push_sustainable_catalyst_feature_suggestions_v3_1_0.sh

To provide an existing repository path explicitly:

bash ./install_and_push_sustainable_catalyst_feature_suggestions_v3_1_0.sh "/path/to/sustainable-catalyst-feature-suggestions"

The script is compatible with the macOS system Bash and does not use mapfile.
