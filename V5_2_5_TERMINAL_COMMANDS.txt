Sustainable Catalyst Product Support and Feedback Platform v5.2.5
Terminal commands for macOS

1. Place one of these files in ~/Downloads:
   - sustainable-catalyst-feature-suggestions-v5.2.5-repo.zip
   - sustainable-catalyst-product-support-feedback-platform-v5.2.5-release-bundle.zip

2. Run the validated install-and-push workflow:

cd ~/Downloads
chmod +x install_and_push_v5_2_5_macos.sh
./install_and_push_v5_2_5_macos.sh

The installer will:
- locate the newest v5.2.5 repository ZIP or release bundle;
- verify the release manifest and compatibility identity;
- clone the current GitHub main branch;
- create a safety backup in ~/Downloads;
- replace the repository working tree while preserving .git;
- create an isolated Python environment;
- run PHP, WordPress contract, JavaScript, JSON, Python, CSS, and FastAPI validation;
- create the v5.2.5 Git commit; and
- push main to GitHub.

Optional: validate an extracted repository without pushing:

cd ~/Downloads/sustainable-catalyst-feature-suggestions-v5.2.5-repository
chmod +x validate_v5_2_5.sh
./validate_v5_2_5.sh

Optional WordPress ZIP installation with WP-CLI:

cd ~/Downloads
wp plugin install sustainable-catalyst-feature-suggestions-v5.2.5-wordpress.zip --force --activate
wp rewrite flush
wp cache flush

Verification:

wp plugin get sustainable-catalyst-feature-suggestions --fields=name,status,version
wp post list --post_type=page --name=knowledge-base --post_status=publish --fields=ID,post_title,post_name,post_parent
wp post list --post_type=sc_support_article --post_status=publish --format=count
wp rewrite list | grep -E 'support/(knowledge-base|guides)|support-documentation'

Open in a browser:
https://sustainablecatalyst.com/support/knowledge-base/
Packaging repair note:
Use the newly downloaded/rebuilt artifacts and remove older v5.2.5 copies from ~/Downloads if needed. The installer selects the newest matching archive by modification time.
