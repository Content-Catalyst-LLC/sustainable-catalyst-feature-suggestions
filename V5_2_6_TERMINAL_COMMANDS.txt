Sustainable Catalyst Product Support and Feedback Platform v5.2.6
Unified Support Center, Embedded Knowledge Base Browser, and Legacy Knowledge Base Route Consolidation

1. Place these files in ~/Downloads:
   - sustainable-catalyst-product-support-feedback-platform-v5.2.6-release-bundle.zip
   - install_and_push_v5_2_6_REPAIRED_macos.sh

2. Run:

cd ~/Downloads
chmod +x install_and_push_v5_2_6_REPAIRED_macos.sh
./install_and_push_v5_2_6_REPAIRED_macos.sh

Python validation requirement:
- Use Python 3.12 or Python 3.13.
- The repaired installer rejects Python 3.14 before dependency installation.
- If needed, run: brew install python@3.13

The installer will:
- locate the newest v5.2.6 repository ZIP or release bundle in ~/Downloads;
- verify the release identity and backward-compatibility manifest;
- clone the current GitHub main branch;
- create a timestamped safety backup in ~/Downloads;
- install the v5.2.6 source tree;
- create an isolated Python validation environment;
- run PHP, WordPress contract, JavaScript, JSON, Python, CSS, whitespace, and FastAPI validation;
- run git diff --check;
- commit the validated release; and
- push the commit to GitHub main.
