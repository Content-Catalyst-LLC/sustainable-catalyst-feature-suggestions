SUSTAINABLE CATALYST PRODUCT SUPPORT AND FEEDBACK PLATFORM v7.0.1
REPOSITORY IDENTITY MIGRATION

1. In GitHub, rename the repository to:
   sustainable-catalyst-product-support-feedback

2. Put the release bundle and installer in ~/Downloads.

3. Run:

cd ~/Downloads
chmod +x install_and_push_sustainable_catalyst_product_support_feedback_v7_0_1_macos.sh
./install_and_push_sustainable_catalyst_product_support_feedback_v7_0_1_macos.sh

The installer accepts either local repository folder:
- sustainable-catalyst-product-support-feedback
- sustainable-catalyst-feature-suggestions

The WordPress plugin folder remains:
- sustainable-catalyst-feature-suggestions
